# PLANO DE GOVERNANÇA: FLUXAI ACADEMY

## 1. Visão Geral
A **FluxAI Academy** atua como a biblioteca oficial de onboarding, treinamento e orientação executiva do FluxAI OS™. Este documento padroniza a separação estrita entre conteúdo operacional e material de cliente.

## 2. Divisão de Módulos
1. **01 - Primeiros Passos**: Conteúdo hibrido (Todos acessam). Visão global da arquitetura.
2. **02 - Operação Interna**: Conteúdo interno (ADMIN/OPERATOR). Treinamento sobre kanban, CRM, Command Center.
3. **03 - Portal do Cliente**: Conteúdo híbrido (Todos acessam). Mostra como o cliente aprova entregas.
4. **04 - Comercial & Apresentação**: Conteúdo interno (ADMIN/OPERATOR). Pitch comercial do sistema para agências ou clientes Enterprise em reuniões.

## 3. Segurança de Dados
- **Uso de Mocks**: Nenhum vídeo será gravado com dados de faturamento reais, nomes de clientes reais, ou tokens/webhooks visíveis no Supabase/Make.
- **Isolamento do DOM**: Extensões ou abas sensíveis do navegador não poderão aparecer.
- **Visualização**: Clientes nunca devem ver módulos operacionais (Módulo 02 e 04).

## 4. Estado Inicial ("GRAVAÇÃO PENDENTE")
Até a captação audiovisual final (Fase 05), a UI do OS exibe um aviso claro de "Gravação Pendente". Nenhuma quebra visual de player ou link morto foi mantida no ar.
