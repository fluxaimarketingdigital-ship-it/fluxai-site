# BLOCO 03: ROADMAP IA DINÂMICO POR CLIENTE

**Status:** Definido e Planejado
**Data:** 05/06/2026

## 1. Visão Estratégica
O Roadmap Operacional IA abandona o formato de "texto fixo universal". Ele passa a ser um artefato inteligente, gerado dinamicamente com base nos dados reais coletados no Onboarding do cliente. O motor de geração cruza o escopo contratado, os canais ativos e as métricas financeiras para construir uma narrativa coerente e hiper-personalizada.

## 2. Estrutura do Motor de Geração
O Roadmap manterá uma estrutura de blocos padrão, mas o preenchimento será condicional:

- **Objetivo:** Injetado via objetivo_principal + segmento.
- **Acessos e Pendências:** Mapeado via canais ativos (Instagram, Meta Ads, WhatsApp, etc).
- **Focos de Narrativa:** Cruzamento de dores_icp, objeções, 	om_de_voz e pilares_editoriais.
- **Semana 1-2 (Setup e Posicionamento):** Ativado apenas para os módulos recém-contratados.
- **Semana 3-4 (Tração Controlada):** Escalonamento das ações operacionais.
- **Semana 5-8 (Otimização) e Semana 9-12 (Consolidação):** Condicionado à existência de verba (erba_midia) e integração de dados (modo_coleta).
- **Governança:** Avisos legais dinâmicos (ex: se elatorio_incluir = nao, alerta que o cliente não tem consolidação mensal).

## 3. Regras de Supressão (O que NÃO prometer)
O algoritmo de IA cortará sumariamente do texto:
- Promessas de Ads se Tráfego Pago não estiver no escopo.
- Criação de Pautas via IA se o módulo Automação & IA não foi comprado.
- Integração via API se o cliente for modo_coleta = manual.

## 4. Tipologias de Cliente
- **FLUXAI_LABS_001 (Interno):** O gerador usa linguagem de "Validação Operacional". Bloqueia termos de cobrança ou envio automático para cliente.
- **Externo Ativo:** Linguagem comercial e focada no SLA.
- **Encerrado:** Roadmap congelado/bloqueado. Apenas visualização do histórico.
- **Teste/Sandbox:** Tarja d'água de teste e bloqueios de disparo.
