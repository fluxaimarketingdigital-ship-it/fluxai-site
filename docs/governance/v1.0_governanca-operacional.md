## 4. Fluxo de Deployment

1.  **Ambiente Local:** Desenvolvimento inicial.
2.  **Staging (Preview):** Validação funcional.
3.  **Produção (Main):** Código homologado.
4.  **Rollback:** Reversão imediata via painel Vercel se necessário.

---

## 5. Padrões de Segurança

*   **Rotação de Senhas:** Semestral para contas críticas.
*   **Tokens & Secrets:** Armazenados exclusivamente em Environment Variables.
*   **Privilégio Mínimo:** Concessão apenas do acesso necessário para cada função.
