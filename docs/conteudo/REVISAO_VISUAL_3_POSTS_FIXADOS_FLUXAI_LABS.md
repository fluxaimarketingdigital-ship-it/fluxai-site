## 🛡️ 4. Diretrizes de Proteção e Sigilo de Ativos (Compliance)

*   **Mockups Sanitizados:** Todas as interfaces renderizadas nos carrosséis utilizam estritamente o cliente fictício de homologação `FLUXAI_LABS_001` com dados quantitativos purificados, livre de chaves Supabase, webhooks Make ou dados de marcas reais.
*   **Controle de Fontes:** Padronização absoluta utilizando *Outfit* para headlines grandes e *Inter* para corpos de textos, garantindo leitura limpa em telas mobile (resolução de feed do Instagram).
*   **Rascunhos Internos:** Os arquivos permanecem salvos em diretório local privado para validação interna da banca antes do agendamento manual de publicação.
