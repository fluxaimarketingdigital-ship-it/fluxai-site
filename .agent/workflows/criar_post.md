# Workflow de Criação de Conteúdo da FluxAI

Bem-vindo ao Workflow de Geração de Posts. 
Sempre que o usuário pedir para criar um post, siga ESTRITAMENTE estas etapas:

## Regras de Tom de Voz da FluxAI
- O tom deve ser **sofisticado, autoritário, persuasivo (mas não agressivo) e focado em alto valor (High-Ticket)**.
- Use emojis moderadamente (referências de luxo: 💎🚀🔥📈🎯).
- Fuja de palavras clichês de marketing barato ("arrasta pra cima", "quer vender mais?"). Substitua por linguagem corporativa avançada ("eleyar o padrão", "escala previsível", "posicionamento dominante").

## Passo 1: O Definição do Formato
Entenda pelo prompt do usuário se o post é de:
- **Imagem Única** (Texto direto ao ponto).
- **Carrossel** (Passo a passo ou educação profunda).
- **Reels/Vídeo** (Gancho muito forte nos primeiros 3s e roteiro dinâmico).

## Passo 2: Geração da Copy (Legenda)
Crie a legenda (Copy) contendo obrigatoriamente:
1. **Headline Chamativa:** O título (primeira linha).
2. **Corpo / Argumentação:** Gatilhos mentais (autoridade, prova, antecipação).
3. **CTA (Chamada para Ação):** Chamando para o link da bio ou Direct para "diagnóstico de alto valor".
4. **Hashtags:** Sugira de 5 a 10 hashtags (Ex: #marketingdigital #brandingpremium #gestaodetrafego #fluxai).

## Passo 3: Sugestão Visual / Prompt de IA
- Escreva um prompt detalhado em INGLÊS que o usuário possa colar no Midjourney ou na ferramenta de IA nativa para gerar as imagens de fundo do post.
- Pense em ambientes sofisticados (tons dourados, grafite, escritórios high-tech minimalistas, profissionais usando computadores modernos ou elementos abstratos 3D elegantes).

## Passo 4: Atualização do CRM
- Escreva o resultado na tela para o usuário aprovar.
- Depois de escrever e sugerir a arte, atualize o quadro Kanban do usuário (o arquivo `c:\Users\BRENDA\Desktop\Identidade Visual FluxAI\Fluxai-site\FluxAI_CRM\CRM_Dashboard.md`), movendo de "Ideias" para "Em Produção" ou "Prontos para Postar".
- Salve o texto final desse post em um arquivo dentro de `c:\Users\BRENDA\Desktop\Identidade Visual FluxAI\Fluxai-site\FluxAI_CRM\Posts_Prontos\nome_do_post.md`.
